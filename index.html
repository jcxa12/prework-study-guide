HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

HTML Elements

  • "p" = defines a paragraph on text
  • "a" = creates hyperlink to another web pages or resources
  • "ul" and "ol" = create unordered and ordered lists, respectively
  • "li" = represents list of items within "ul" and "ol" elements
  • "span" = a generic container for grouping and styling content

HTML Attributes

  • "href" (in "a") = specifies the URL or resource to link to
  • "src" (in "img") = defines the source URL for an image
  • "alt (in "img") = provides alternative text for images for accessibility
  • "id" = gives a unique identifier to an element, often used for CSS or JavaScript
  • "class" = assigns one or more classes to an element for styling purposes
  • "style" = allows inline CSS styling for individual elements
  • "target" (in "a") = specifies where the linked content will open
  • "type" (in "script" and "link") = defines the type of resources being linked or embedded

Git

  • git init - "Start a Magic Notebook": It's like starting a special notebook for your drawings.
  • git clone - "Copy Friend's Notebook": You can make a copy of your friend's notebook with all their drawings.
  • git add - "Pick and Put on Shelf": Choose a drawing and put it on a special shelf so you don't forget it.
  • git commit -m "message" - "Save with Note": When you're done drawing, you save your work and leave a little note to remember what you did.
  • git status - "Check the Shelf": You look at the special shelf to see what's there and what you still need to save.
  • git pull - "Get New Drawings": If your friend has new drawings, you can ask them to share with you.
  • git push - "Give Your Drawings": If you have new drawings, you can share them with your friend.
  • git branch - "Different Magic Notebooks": You can have different notebooks for different kinds of drawings.
  • git checkout - "Change Notebook": You can switch to a different notebook for different drawings.
  • git merge - "Mix Drawings": If you have drawings in two notebooks, you can mix them together.
  • git log - "History Book": It's like a book that tells you a story about all the drawings you made.
  • git stash - "Hide Drawings": Sometimes you want to hide your drawings for a while and then bring them back.
  • git remote -v - "Friends' Notebooks": You can see if your friends have notebooks like yours.
  • git checkout -b branch-name: creates a new branch and switches to it
